# Scalera AI

Scalera AI is an enterprise-grade AI website generator that builds professional, high-fidelity websites in seconds.

## Features
- **AI-Driven Personalization**: Deep rewrite of template content using Groq LLMs.
- **Enterprise Templates**: 10+ premium templates for various industries.
- **Fast & Lightweight**: Built with performance and minimalist aesthetics in mind.

## Setup
1. Clone the repository.
2. Install dependencies: `pip install -r backend/requirements.txt`.
3. Set your `GROQ_API_KEY` environment variable.
4. Run the backend: `python backend/main.py`.

## Deployment
Deployed on Vercel with automatic syncing from GitHub.
